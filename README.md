## Bhawa Mobile App
> This mobile app is developed using the React-native Framework for the Bhawa talent show.

#### Features
1) Login / Register
2) Event schedule
3) Live streaming
4) Voting
5) Leaderboard
6) Setting
7) About
8) Ticketing
9) User friendly
